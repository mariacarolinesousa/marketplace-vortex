# Requisitos do Projeto

## Objetivo

Desenvolver uma Progressive Web App (PWA) para um Marketplace de Economia Circular voltado para estudantes da universidade.

## Funcionalidades Obrigatórias

- Landing Page
- Cadastro de anúncios
- Listagem de anúncios
- Edição de anúncios
- Exclusão de anúncios
- API REST
- Banco de dados
- Responsividade
- PWA

## Tecnologias

- React
- Node.js
- Express
- Prisma
- PostgreSQL
- TypeScript
