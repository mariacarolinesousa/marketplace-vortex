# Planejamento do Projeto

## Nome do Projeto

EcoCampus - Marketplace de Economia Circular

## Objetivo

Desenvolver uma plataforma web/ aplicação mobile pwa que permita aos estudantes da universidade anunciar itens para venda ou doação, incentivando a reutilização de materiais e promovendo a economia circular no campus.

## Público-alvo

- Estudantes universitários
- Professores
- Comunidade acadêmica

## Problema

Muitos estudantes possuem livros, calculadoras, componentes eletrônicos, jalecos e outros materiais que deixam de utilizar, enquanto outros alunos precisam desses itens.

O sistema permitirá anunciar esses materiais para venda ou doação, reduzindo desperdícios e facilitando o acesso de produtos à alunos que estão iniciando o curso

## Funcionalidades Principais

Visitante

- Visualizar anúncios
- Filtrar anúncios por categoria
- Conhecer o projeto

Usuário

- Criar anúncio
- Visualizar seus anúncios
- Excluir anúncio
